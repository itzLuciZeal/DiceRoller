# Dice Rolling Game
An **interactive web-based dice roller** designed for tabletop games, featuring **customizable dice skins**. **[Live Demo](https://itzlucizeal.github.io/DiceRoller/)**

![GitHub last commit](https://img.shields.io/github/last-commit/itzLuciZeal/DiceRoller)
![GitHub License](https://img.shields.io/github/license/itzLuciZeal/DiceRoller)


## Features
> Here are the key features you would expect when you visit the project:
 - **Customizable Dice Skins**: You can customize your own likings of your favorite dice skins when you roll. Though it is limited only and can be requested to me to add it.
 - **Customizable Accent Colors & Themes**: You can customize your own accent colors that is align with your personality, as well as the theme colors.
 - **Responsive UI for Desktop and Mobile**: I made it responsive not only in desktops but to all devices including mobile.
 - **Documentation of Journey**: I have documented how I made this possible, the problems I encounter and also how I solved them.

## How to Use
> You can just go ahead to the very bottom of the website, and start clicking the `roll dice` button.
 - **Settings Page**: In the top-right header, you can click the **gear icon** that will lead you to the `settings` page where you can edit your **themes**, **accent colors**, and **dice skins**.

## For Local Development
> If you want to try it out locally in your device you can follow these simple steps to do so, assuming you are using laptop or desktop:
 - **Clone the Repository**: You can just paste this code block in your terminal `git clone https://github.com/itzLuciZeal/DiceRoller.git` so you can copy it to your devices storage.
 - **Start Local Hosting**: In **Visual Studio Code**, you can just right click the `index.html` and run Live Server extension.
